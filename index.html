Simple Dockerized Web App

Step-by-step guide to containerize a web application

My Dockerized Web Application

This is a simple example of a web application running inside a Docker container.

Current time:

Container hostname: localhost

Step-by-Step Process

1 Create Project Structure

Create these files in your repository:

my-app/ ├── index.html ├── Dockerfile ├── .dockerignore ├── .github/ │ └── workflows/ │ └── deploy.yml └── README.md

2 Create index.html

Create a simple HTML file:

<!DOCTYPE html>
<html>
<head>
    <title>My App</title>
</head>
<body>
    <h1>Hello from Docker!</h1>
</body>
</html>

3 Create Dockerfile

Create a Dockerfile to build your image:

# Use the official Nginx image
FROM nginx:alpine

# Copy your HTML file
COPY index.html /usr/share/nginx/html/index.html

# Expose port 80
EXPOSE 80

# Start Nginx
CMD ["nginx", "-g", "daemon off;"]

4 Build and Test Locally

Build and run your Docker container:

# Build the image
docker build -t my-web-app .

# Run the container
docker run -d -p 8080:80 --name web-container my-web-app

# View in browser
open http://localhost:8080

5 Create CI/CD Pipeline

Create .github/workflows/deploy.yml:

name: Deploy to Docker Hub

on:
  push:
    branches: [ main ]

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
    - uses: actions/checkout@v2
    - name: Build Docker image
      run: docker build -t my-web-app .
    - name: Test Docker image
      run: |
        docker run -d --name test-container my-web-app
        docker ps | grep test-container
        docker stop test-container

6 Push to GitHub

Push your code to trigger the pipeline:

# Add files to git
git add .

# Commit changes
git commit -m "Add Docker setup"

# Push to GitHub
git push origin main

Note: This is a simplified example. In a real-world scenario, you would also want to:

  • Add a .dockerignore file to exclude unnecessary files
  • Set up Docker Hub or another container registry
  • Add proper error handling and testing
  • Configure environment variables for different deployment environments

This guide provides a basic overview of Dockerizing a web application.

For production use, refer to the official Docker and GitHub Actions documentation.
